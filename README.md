# Python-скрипт для записи стримов на Twitch
Далеко не все стримеры оставляют запись после завершения трансляции, поэтому этот скрипт поможет вам вести запись трансляции, чтобы посмотреть её потом в удобное время.
## Пример `.env` файла
Скрипт работает с переменными окружения из файла `.env`
```
CHANNEL=https://www.twitch.tv/name
QUALITY=best
CHECK_INTERVAL=30

RECORDS_DIR=/data

HTTP_SERVER_ENABLED=false
SERVER_HOST=0.0.0.0
PORT=8080
```
`QUALITY` позволяет задать качество записи стрима. Примеры значений: `1080p60`, `1080p`, `720p60`, `720p`, `best`.  
Если указанное качество отсутствует у стрима, скрипт автоматически покажет список доступных качеств и переключится на `best`.

`CHECK_INTERVAL` - интервал между проверками начала стрима в секундах.  

Значение переменной `HTTP_SERVER_ENABLED` позволяет запустить HTTP сервер.  
Установите значение `true` (или `yes`, `on`), **если у вас есть в нём необходимость**, в случае ненадобности установите значение `false`.

`SERVER_HOST` и `PORT` используются для HTTP сервера. Значения можно оставить по умолчанию, как в примере.

Думаю, переменные `CHANNEL` и `RECORDS_DIR` не нуждаются в комментариях, всё и так очевидно.
## Использование без Docker
Если вы не используете Docker для запуска этого скрипта, убедитесь, что у вас установлен ffmpeg.  
Для автоматической установки всех необходимых зависимостей для Python используйте команду  
```
pip install -r requirements.txt
```  

## Конвертация `.ts` в `.mp4`
По умолчанию стрим записывается в формате TS (Transport Stream), что для некоторых может быть неудобным.  
Используя ffmpeg вы можете конвертировать запись стрима в привычный всем MP4 с помощью вот этой команды:  
```
ffmpeg -i stream_filename.ts -c copy -movflags +faststart stream_converted.mp4
```
где `stream_filename.ts` - это файл (запись стрима) в формате TS, а `stream_converted.mp4` - это выходной файл в формате MP4.  

💡 Перекодирование не производится (используется `-c copy`). Команда лишь меняет обертку файла, поэтому выполняется очень быстро и практически не потребляет ресурсы CPU/GPU.
