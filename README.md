## Пример `.env` файла
Python-скрипт работает с переменными окружения из файла `.env`
```
CHANNEL=https://www.twitch.tv/name
QUALITY=best
CHECK_INTERVAL=30

RECORDS_DIR=/data

HTTP_SERVER_ENABLED=true
SERVER_HOST=0.0.0.0
PORT=8080
```
Примеры значений для `QUALITY`: `1080p60`, `1080p`, `720p60`, `720p`, `best`.  
Если указанное качество отсутствует у стрима, скрипт автоматически покажет список доступных качеств и переключится на `best`.

`HTTP_SERVER_ENABLED` позволяет запустить HTTP сервер.  
Установите значение `true` (или `1`, `yes`, `on`), **если у вас есть в нём необходимость**, в случае ненадобности установите значение `false`.

## Примечание
Если вы не используете Docker для запуска этого скрипт, убедитесь, что у вас установлен ffmpeg.
